CrowdStrike Falcon Deployment Automation

Designed and implemented an automated deployment pipeline for CrowdStrike Falcon Sensor across more than 4,000 Linux servers spanning multiple distributions and Puppet versions.

The solution used AWX, Puppet, Artifactory, and SHA256-based package handling to standardize endpoint security deployment, reduce manual installation effort, and ensure controlled rollout of approved sensor versions.

OpenVPN Access Server for Vendor Segmentation

Designed and implemented a secure OpenVPN Access Server platform for external vendor access, integrating Okta authentication with local administrative separation and vendor-specific network boundaries.

The platform reduced third-party access risk by limiting vendors to approved internal resources, while custom Icinga monitoring improved service visibility and reduced noisy alerts.

JFrog Artifactory and Xray Platform

Built and supported a multi-environment artifact management and security scanning platform using JFrog Artifactory, Xray, PostgreSQL, federated repositories, and controlled repository promotion workflows.

The platform improved build reliability, reduced external dependency risk, enabled software supply chain visibility, and established a scalable foundation for artifact governance and vulnerability scanning.

Zabbix Monitoring Standardization

Implemented standardized monitoring for Linux, SNMP-enabled infrastructure, and Windows systems using Zabbix, PostgreSQL, Puppet automation, SNMP traps, and Group Policy deployment.

The work improved monitoring consistency, centralized operational visibility, reduced manual configuration, and created repeatable patterns for infrastructure health checks and alerting.
